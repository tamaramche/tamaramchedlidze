Point Set Embeddings of Planar Graphs

10 April 2018

Point Set Embedding is a classical problem in the intersection of graph theory and computational geometry. Given a planar graph G=(V,E) and a set of points in the plane S, with |V|=|S|, the problem asks whether there exists a bijective mapping from V to S, such that the implied straight-line drawing of G is planar. The Point Set Embedding problem is rather of theoretical interest and does not seem to be contributing to solving any real-world problem. Despite that, the non-trivial nature of the problem keeps attracting researchers attention since 1990, when the problem was presented for rooted trees by Perles at DIMACS workshop on arrangements. One of the basic results on this problem is that every rooted tree admits an embedding on every set of points in general position. The result remains true even if the root of the tree has to lie on a specific point of the convex hull (Perles), on the deepest point of the point set (Pach et al.) and when the root of the tree is constrained to any point of the point set (Ikebe et al.).

I have studied several variations of the Point Set Embedding problem: where the point set is larger than the vertex set of the graph, thus a subset of points has first to be chosen for an embedding; a version where the point set is smaller than the graph, thus the vertices that are mapped to the points have to be selected; the version where both points and vertices are colored and the task is to find a color-consistent mapping; the version where the graph is directed and the mapping has in some sense to respect the direction of the edges; and finally, where instead of straight-lines, circular edges are used for the display of the edges.
